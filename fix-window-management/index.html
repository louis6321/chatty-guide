Fixing Window Management

Chatty's dark skins don't support window management (like snapping) within Windows.
Follow these steps to fix that.
MAKE SURE YOU HAVE THE LATEST BETA VERSION OF CHATTY!



Start by executing this command in Chatty:
/set lafCustomTheme windowDecoration off

Restart Chatty and window management should now be working, but the title bar will either be white or a system colour, so let's fix that.


Open your start menu and search for "accent color".
Select the "Choose your accent color" result.



Tick the "Title bars and window borders" option at the bottom, then close the window.



Finally, run this reg file: chatty.reg
You will then need to logout and log back in (or reboot) for this change to apply.
